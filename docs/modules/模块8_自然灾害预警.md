### 1. 模块定位
融合**气象数据、河道水位监测、积水点监测、无人机河道巡检**等多源信息，对潜在自然灾害（暴雨、台风、洪水、内涝等）进行实时预警，形成**数据采集 → 阈值判断 → 预警生成 → 派发通知 → 应急响应**的管理流程。一级页面突出**当前活跃预警**（超警戒水位/积水/气象预警）及**河道/积水实时数据**，驱动值班人员立即通知相关单位防范；二级页面支持历史水位趋势、预警响应评估及无人机巡检记录分析。

+ **使用人群**：镇政府值班人员/防汛抗旱指挥人员 → 一级页面突出“水位超警戒”“积水预警”“气象预警”，驱动快速发布防范指令。
+ **数据显示目的**：告诉使用者“哪些地方有洪涝/积水风险，需立即处置” → 设置“最新预警列表”按风险等级排序，支持一键通知责任单位。
+ **统计口径**：一级页面默认**实时**数据（水位、积水深度每5分钟更新）及**当前生效中的预警**（气象部门发布的预警信号）；二级页面支持按日期、点位、预警类型筛选全部历史数据。

### 2. 一级页面核心数据（主屏概览）
| 核心指标 | 统计口径 | 展示形式 | 目的与行动含义 |
| --- | --- | --- | --- |
| 河道超警戒点数 | 辖区内所有河道监测点中，当前水位超过警戒水位的点位数量（不区分预警等级） | 大号橙色数字，标注“需关注” | 告知值班人员当前哪些河道有漫堤风险，驱动立即巡查或泄洪调度 |
| 积水预警点数 | 辖区内低洼易涝点中，当前积水深度超过预警阈值（如≥15cm）的点位数量 | 大号橙色/红色数字（根据严重程度），标注“需疏导” | 快速定位内涝风险点，驱动安排排水或交通管制 |
| 气象预警等级 | 当前生效中的最高气象预警信号（台风/暴雨/雷雨大风等），从气象部门接口获取 | 颜色图标（红/橙/黄/蓝）+ 预警类型文字 | 直观了解整体天气威胁等级，决定是否启动应急响应 |
| 无人机巡检异常数 | 最近一次（或今日）无人机河道巡检中发现的异常数量（如堤坝裂缝、堵塞、违占等） | 数字 + “需复核”图标 | 及时发现河道安全隐患，驱动人工复核或清理 |
| 最新预警列表 | 按**预警等级降序（红>橙>黄>蓝）** 及**时间倒序**展示前5条预警，每条显示：预警类型（水位/积水/气象/无人机异常）、位置/区域、当前数值/阈值、预警等级、发布时间 | 列表，每条右侧带“通知责任人”按钮 | 快速定位最严重的预警，一键通知对应村/社区/单位防范 |


> **模块标题旁**：可显示“数据实时刷新”小标签，并设“查看全部历史”链接跳转二级页面。地图集成显示各监测点位（颜色标识预警状态）。

### 3. 二级专题页面核心数据（点击下钻后）
进入“自然灾害预警”专题页，提供**时间筛选器**（今日/近7天/近30天/自定义）、**点位筛选**（河道/积水点/无人机航线）、**预警类型筛选**（水位/气象/积水/无人机异常），分为四大区块：

#### 3.1 综合统计看板（随筛选条件变化）
+ 预警事件总数（按条统计，每次水位越限、积水超阈值、气象发布均计为一次预警事件）
+ 超警戒累计时长（所有河道水位超警戒的总分钟数）
+ 最大积水深度（时间范围内出现的最大值及位置）
+ 无人机巡检次数 & 发现异常数 & 异常处置率
+ 各类预警数量分布（水位超警/积水预警/气象预警/无人机异常）
+ 平均预警响应时长（从预警产生到首次通知责任单位的时间）

#### 3.2 预警事件列表（支持筛选、排序）
+ 表格字段：预警时间、预警类型、位置/点位、预警等级（红/橙/黄/蓝）、当前数值（水位/积水深度等）、阈值、状态（已处置/未处置）、通知对象
+ 支持按预警时间、预警等级、状态排序和筛选
+ 每条预警右侧可操作：**查看详情**（下钻到预警明细）、**通知责任人**（一键发送短信或App推送）

#### 3.3 单条预警明细页（点击预警详情后）
+ **预警基本信息**：
    - 预警类型、发生时间、位置（地图打点）
    - 预警等级、触发条件（如“水位3.2m，警戒线2.5m”）
    - 数据来源（水位计/气象局/无人机/人工巡查）
    - 现场图片/视频（如有，如无人机拍摄的堤坝裂缝照片）
+ **预警处置记录**：
    - 通知时间、通知对象（村支书/水利站/应急队）、通知方式（短信/电话/App）
    - 响应措施：如开闸泄洪、组织巡查、设置警戒带、排水泵启动等
    - 处置结果：是否解除预警、解除时间
+ **历史水位/积水曲线**：展示预警前后12小时的水位/积水深度变化，便于评估趋势。
+ **操作**：重新通知责任单位、标记已处置、上传现场照片、生成预警报告（PDF）

#### 3.4 历史趋势与效率分析（折线图/柱状图）
+ **按月/日展示**：
    - 各河道月均水位与警戒线对比（或超警戒天数）
    - 各易涝点积水预警频次排名
    - 气象预警数量与水位/积水预警的相关性（降雨量 vs 水位涨幅）
+ **无人机巡检趋势**：
    - 每月巡检里程/次数、发现异常数、异常类型分布（裂缝/堵塞/违占/其他）
    - 异常处置率变化
+ **响应效率**：
    - 平均预警响应时长按月对比
    - 各责任单位响应速度排名
