### 1. 模块定位
基于**监控视频**与**AI算法**，对辖区内的安全、消防、应急等场景进行实时智能分析，自动识别异常事件并生成预警，形成**视频监控 → AI识别 → 预警生成 → 派发处置 → 闭环反馈**的管理流程。一级页面突出**今日未处理预警事件**及**实时视频**，驱动值班人员立即核实与处置；二级页面支持历史预警追踪、AI识别准确率分析及事件处置效率评估。

+ **使用人群**：镇政府值班人员/应急指挥人员 → 一级页面突出“实时视频+未处理预警”，驱动快速响应。
+ **数据显示目的**：告诉使用者“现在有哪些AI预警需要处理” → 设置“最新预警列表”按时间倒序，未处理高亮，支持一键派单。
+ **统计口径**：一级页面默认**今日（0点至今）** 产生的预警事件（未处理/处理中/已闭环），视频流实时刷新；二级页面支持按时间、点位、预警类型筛选全部历史数据。

### 2. 一级页面核心数据（主屏概览）
| 核心指标 | 统计口径 | 展示形式 | 目的与行动含义 |
| --- | --- | --- | --- |
| 今日未处理预警数 | 今日产生的预警事件中，状态为“未处理”或“处理中”的事件总数 | 大号红色数字，标注“需处置” | 告知值班人员今日尚有未闭环的预警，驱动立即核实现场 |
| 今日预警总数 | 今日产生的所有预警事件总数（含已处理） | 数字（灰色背景），旁标环比昨日同期变化 | 宏观了解今日AI预警活跃程度，判断是否异常增多 |
| 各类型预警分布 | 按预警类型（火情/烟雾/违规闯入/异常聚集/未戴安全帽等）统计今日未处理数量 | 水平条形图或饼图，高占比类型突出显示 | 快速识别高频预警类型，针对性派发或检查设备 |
| 实时监控视频流 | 重点点位（如主要厂房、河道、学校门口等）的实时视频，支持轮播或手动切换 | 视频窗口（可1主多辅），右下角叠加最近预警标记 | 提供直观现场画面，辅助判断预警真实性及紧急程度 |
| 最新预警列表 | 按**预警时间倒序**展示前5条未处理/处理中的预警，每条显示：预警类型、发生点位、预警时间、当前状态、AI置信度 | 列表，每条右侧带“处理”按钮 | 快速定位到最新预警，一键进入处置流程 |


> **模块标题旁**：显示“数据范围：今日”小标签，并设“查看全部历史”链接跳转二级页面。视频窗口下方可设“切换点位”下拉框。

### 3. 二级专题页面核心数据（点击下钻后）
进入“安消应AI预警”专题页，提供**时间筛选器**（今日/近7天/近30天/自定义）、**点位筛选**（按摄像头名称或区域）、**预警类型筛选**（火情/烟雾/闯入等）及**状态筛选**（未处理/已处理/已闭环），分为四大区块：

#### 3.1 综合统计看板（随筛选条件变化）
+ 预警事件总数
+ 未处理数 & 处理中数 & 已闭环数
+ 平均响应时长（从预警生成到首次派发/确认的时间，仅统计已处理）
+ 平均处置时长（从预警生成到闭环的时间，仅统计已闭环）
+ AI识别准确率（已验证的准确预警数 / 总预警数 ×100%，依赖于后续人工反馈）
+ 高频预警类型Top3

#### 3.2 预警事件列表（支持筛选、排序）
+ 表格字段：预警时间、预警类型、发生点位（摄像头名称）、预警截图/短视频（缩略图）、状态（未处理/处理中/已闭环）、AI置信度、处置人
+ 支持按预警时间、置信度、状态排序和筛选
+ 每条预警右侧可操作：**查看详情**（下钻到预警明细）、**处理**（派发或标记状态）

#### 3.3 单条预警明细页（点击预警详情后）
+ **预警信息**：预警时间、预警类型、发生点位、AI置信度、预警截图（或短视频片段）、原始监控视频回放链接
+ **AI识别细节**：识别的目标框、异常行为描述（如“火光检测”“人员倒地”等）
+ **处置记录**：
    - 是否派发：派发时间、接收单位/人、联系方式
    - 现场核实：核实人、核实时间、核实结果（误报/真实事件）
    - 处置动作：采取的措施（如呼叫消防、疏散人员）、处置完成时间
    - 闭环确认：闭环时间、备注
+ **操作**：重新派发、标记误报、上传现场照片/视频、标记闭环
+ **导出**：导出预警报告为PDF（含截图、处置记录）

#### 3.4 历史趋势与效率分析（折线图/柱状图）
+ **按日/周展示**：
    - 每日预警总数 vs 误报数（判断AI稳定性）
    - 每日平均响应时长趋势（考核处置效率）
    - 各点位预警数量排名（找出事件高发区域）
+ **预警类型分析**：
    - 不同预警类型的数量占比及处置完成率
    - 不同点位的预警类型分布（如河道点位多为“人员闯入”，厂房点位多为“烟雾”）
+ **位点健康度**：
    - 摄像头在线率、离线时长（确保监控覆盖有效）
