### 1. 模块定位
围绕**疫情相关数据上报**（如发热病例、聚集性症状、传染病报告等），实现**表单上报 → 数据审核 → 生命周期跟踪 → 应急预案匹配**的全流程管理，辅助镇政府及时发现疫情苗头、启动响应。一级页面聚焦**今日新增上报、待审核/待处置的异常上报**以及**已匹配应急预案的上报**，驱动值班人员快速审核、派发处置；二级页面支持历史上报数据查询、生命周期追溯、应急预案匹配效果分析。

+ **使用人群**：镇政府防疫值班人员/卫健办工作人员 → 一级页面突出“待审核上报数”和“已匹配应急预案的异常上报”，驱动快速响应。
+ **数据显示目的**：告诉使用者“哪些上报需要审核、哪些已触发预案需处置” → 设置“最新上报列表”按紧急状态（待审核/处置中/已匹配预案）排序，支持一键匹配预案。
+ **统计口径**：一级页面默认**今日（0点至今）** 新上报数据 + 所有状态为“待审核”“处置中”且未闭环的上报（不限时间）；二级页面支持按日期、上报类型、生命周期状态筛选全部历史数据。

### 2. 一级页面核心数据（主屏概览）
| 核心指标 | 统计口径 | 展示形式 | 目的与行动含义 |
| --- | --- | --- | --- |
| 今日新增上报数 | 今日0点至今新增的上报表单总数（含所有状态） | 数字（灰色背景），旁标环比昨日同期变化 | 了解当日疫情监测活跃度，判断是否异常增多 |
| 待审核上报数 | 状态为“待审核”的上报总数（不限时间，重点突出近期） | 大号橙色数字，标注“需审核” | 告知值班人员有未审核的上报，需尽快核实真实性 |
| 已匹配预案数（处置中） | 已匹配应急预案且尚未闭环的上报数量（状态为“预案匹配中/处置中”） | 红色数字 + 警示图标 | 提示需跟踪预案执行进度，防止疫情扩散 |
| 异常上报率 | 今日新增上报中被标记为“异常”（如聚集性症状、确诊关联等）的比例 | 百分比 + 趋势箭头 | 宏观判断疫情风险趋势 |

> **模块标题旁**：显示“数据范围：今日+待处置”小标签，并设“查看全部历史”链接跳转二级页面。

### 3. 二级专题页面核心数据（点击下钻后）
进入“疫情监测预警”专题页，提供**时间筛选器**（今日/近7天/近30天/自定义）、**上报类型筛选**（发热/腹泻/聚集性症状/传染病报告/其他）、**生命周期状态筛选**（待审核/审核通过/预案匹配中/处置中/已闭环/已驳回），分为四大区块：

#### 3.1 综合统计看板（随筛选条件变化）
+ 上报总数
+ 待审核数 & 处置中数 & 已闭环数
+ 平均审核时长（从上报到审核完成的小时数）
+ 平均处置时长（从上报到闭环的小时数，仅统计已闭环）
+ 异常上报率（标记为异常的上报数 / 上报总数）
+ 匹配应急预案的上报数 & 预案执行完成率

#### 3.2 上报列表（支持筛选、排序）
+ 表格字段：上报时间、上报单位/人、上报类型、简要内容、当前状态（待审核/审核通过/预案匹配中/处置中/已闭环/已驳回）、是否异常、匹配预案名称
+ 支持按上报时间、状态、是否异常排序和筛选
+ 每条上报右侧可操作：**查看详情**（下钻到上报明细）、**审核**（快速审核通过/驳回）、**匹配预案**（手动或自动触发）

#### 3.3 单条上报明细页（点击上报详情后）
+ **上报信息**：
    - 上报人信息（姓名、单位/村社区、联系方式）
    - 上报时间、上报类型
    - 表单字段内容（如发热人数、症状描述、接触史、发生地点等）
    - 附件（图片、Excel等）
+ **审核记录**：
    - 审核人、审核时间、审核结果（通过/驳回/需补充）、审核意见
+ **生命周期跟踪**（时间轴）：
    - 上报 → 审核 → 预案匹配（若触发） → 任务派发 → 现场处置 → 结果反馈 → 闭环
    - 每个节点的时间、操作人、备注
+ **应急预案匹配**：
    - 匹配的预案名称及版本
    - 预案中的响应措施（如隔离、消杀、采样、上报疾控）
    - 预案执行任务列表（派发到哪些部门/人员、是否完成）
+ **操作**：重新审核、重新匹配预案、派发任务、上传处置结果、标记闭环、导出上报报告（PDF/Word）

#### 3.4 历史趋势与效率分析（折线图/柱状图）
+ **按日/周展示**：
    - 每日上报总数 vs 异常上报数
    - 平均审核时长、平均处置时长变化趋势
    - 匹配预案的上报数量趋势
+ **上报类型分析**：
    - 不同上报类型的数量占比及异常率
    - 不同区域/村社区的活跃度排名
+ **预案匹配效果**：
    - 各个应急预案被触发的频次
    - 预案执行完成率及平均处置时长对比
