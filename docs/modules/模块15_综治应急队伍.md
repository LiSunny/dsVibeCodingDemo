### 1. 模块定位
建立**应急队伍台账**（队伍基本信息、人员构成、装备配置、擅长领域等），支持**警情分发调度**（将突发事件或任务派发给相应队伍）。实现**队伍信息管理 → 任务派发 → 响应跟踪 → 统计分析**的闭环管理。一级页面突出**当前可调度队伍数、待分发警情数、队伍分布**，驱动值班人员快速派警；二级页面支持队伍档案管理、调度历史追溯及响应效率评估。

+ **使用人群**：镇政府应急指挥中心/综治办人员 → 一级页面突出“可调度队伍”和“待分发警情”，驱动快速派发任务。
+ **数据显示目的**：告诉使用者“现在有多少队伍可用、有多少警情需要分发” → 设置“队伍列表”按距离/擅长领域排序，支持一键派发。
+ **统计口径**：一级页面默认**所有应急队伍**（含在岗/执勤状态）及**待分发的警情/任务**（来自其他模块如隐患、预警等）；二级页面支持按队伍类型、区域、响应状态筛选全部历史数据。

### 2. 一级页面核心数据（主屏概览）
| 核心指标 | 统计口径 | 展示形式 | 目的与行动含义 |
| --- | --- | --- | --- |
| 当前可调度队伍数 | 状态为“在岗”或“待命”且未在执行任务中的应急队伍总数 | 大号绿色数字，标注“可用” | 告知值班人员有多少队伍可随时调用，便于快速派发任务 |
| 待分发警情数 | 其他模块（如隐患、预警、吹哨人等）产生的、尚未派发给应急队伍的任务/警情总数 | 大号橙色数字，标注“需派发” | 提示值班人员有未分派的任务，需尽快安排队伍处理 |
| 队伍总数 | 辖区内所有注册应急队伍总数 | 数字（灰色背景），旁标按类型统计 | 了解整体队伍规模 |
| 队伍平均响应时长（近30天） | 近30天内派发任务，从派发到队伍确认响应/到达现场的平均时长（分钟） | 数字 + 趋势箭头 | 考核应急队伍的反应速度 |
| 可调度队伍列表 | 按**擅长领域匹配度/距离**展示前5条可调度队伍，每条显示：队伍名称、所属区域、擅长领域（消防/医疗/治安等）、当前人数、在岗状态、距离事发地（估算） | 列表，每条右侧带“派发”按钮 | 快速选择合适队伍，一键派发任务 |


> **模块标题旁**：显示“队伍状态实时刷新”小标签，并设“查看全部历史”链接跳转二级页面。可集成地图展示队伍分布（打点）。

### 3. 二级专题页面核心数据（点击下钻后）
进入“综治应急队伍”专题页，提供**区域筛选**（各村/社区）、**队伍类型筛选**（消防/医疗/治安/综合/其他）、**状态筛选**（在岗/执勤/休整/离线），分为四大区块：

#### 3.1 综合统计看板（随筛选条件变化）
+ 队伍总数 & 总人数
+ 在岗队伍数 & 执勤队伍数 & 休整队伍数
+ 装备配置完整率（有核心装备的队伍数 / 总队伍数）
+ 平均响应时长（近30天派发任务的平均确认响应时长）
+ 任务完成率（完成任务数 / 派发任务总数 ×100%）
+ 各类队伍数量分布

#### 3.2 队伍列表（支持筛选、排序）
+ 表格字段：队伍名称、所属区域、类型、队长、联系方式、当前人数、在岗状态、擅长领域、最近一次任务时间、平均响应时长
+ 支持按在岗状态、平均响应时长、类型排序和筛选
+ 每条队伍右侧可操作：**查看详情**（下钻到队伍明细及成员）、**派发任务**（手动选择该队伍处理某警情）

#### 3.3 队伍档案及任务明细页（点击队伍详情后）
+ **队伍基本信息**：
    - 队伍名称、所属村/社区、类型（消防/医疗/治安/综合）
    - 队长姓名、电话、副队长
    - 注册人数、当前在岗人数
    - 装备配置（车辆、灭火器、急救包、无人机等，支持清单）
    - 擅长领域（如水上救援、山地搜索等）
    - 常驻位置/集结点（地图打点）
+ **成员列表**：
    - 姓名、职务、电话、技能证书（如有）、在岗状态
+ **任务记录**（按时间倒序）：
    - 派发时间、任务描述、任务来源（如隐患模块）、响应时间、到场时间、任务完成时间、完成结果
+ **响应效率分析**（折线图）：近6个月平均响应时长变化
+ **操作**：编辑队伍信息、配置装备、手动派发任务、导出队伍档案（PDF/Excel）

#### 3.4 历史趋势与效率分析（折线图/柱状图）
+ **按日/周展示**：
    - 派发任务数 vs 完成数
    - 平均响应时长趋势
+ **队伍类型分析**：
    - 各类队伍的平均响应时长对比
    - 各类队伍的任务完成率对比
+ **区域分析**：
    - 各村/社区队伍数量及响应效率
    - 任务高发区域与队伍覆盖匹配度
