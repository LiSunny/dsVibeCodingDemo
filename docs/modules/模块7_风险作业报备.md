### 1. 模块定位
对辖区内企业**动火作业、受限空间作业、高空作业**等风险作业进行**报备→审批→过程监督→完成确认**的管理，实现风险作业可查看、可追溯、可预警。一级页面突出**今日正在进行的风险作业、即将超时作业（超过规定时长未完成）**，驱动值班人员实时掌握高风险作业动态，必要时现场巡查；二级页面支持历史作业记录查询、超时原因分析及企业作业合规性评估。

+ **使用人群**：镇政府安监站/值班人员 → 一级页面突出“进行中风险作业”和“超时未完成作业”，驱动关注高风险作业现场安全。
+ **数据显示目的**：告诉使用者“今天有哪些风险作业在进行、哪些已超时需提醒” → 设置“最新作业列表”按紧急状态（超时 > 进行中 > 即将开始）排序，支持一键提醒。
+ **统计口径**：一级页面默认**今日所有状态为“已审批通过”“作业中”的风险作业**，以及**超时未完成作业**（计划结束时间已过但未报完成）；二级页面支持按作业时间、作业类型、企业筛选全部历史数据。

### 2. 一级页面核心数据（主屏概览）
| 核心指标 | 统计口径 | 展示形式 | 目的与行动含义 |
| --- | --- | --- | --- |
| 今日进行中作业数 | 今日状态为“作业中”（已审批通过且计划时间覆盖今日）的风险作业总数 | 大号橙色数字，标注“需关注” | 告知值班人员今日有多少风险作业正在进行，需关注安全措施落实情况 |
| 超时未完成作业数 | 计划结束时间已过，但尚未标记“已完成”的风险作业总数 | 大号红色数字，标注“超时需提醒” | 提醒值班人员联系企业确认作业状态，防止作业失控或违规延长 |
| 今日新增报备数 | 今日0点至今新增的风险作业报备总数（含待审批/已审批） | 数字（灰色背景），旁标环比昨日同期 | 了解当日风险作业活跃程度，判断是否高峰期 |
| 动火作业占比 | 今日进行中作业中，动火作业所占比例 | 百分比 + 环形图 | 动火作业风险高，高占比提示需加强巡查 |
| 最新风险作业列表 | 按**紧急优先级**（超时 > 作业中（计划结束时间近） > 即将开始）展示前5条，每条显示：作业类型、企业名称、作业时间、作业位置、当前状态（作业中/超时）、安全措施确认 | 列表，每条右侧带“提醒”按钮 | 快速定位最紧急的作业（超时或即将到期），一键提醒企业安全负责人 |


> **模块标题旁**：显示“数据范围：今日作业+超时作业”小标签，并设“查看全部历史”链接跳转二级页面。

### 3. 二级专题页面核心数据（点击下钻后）
进入“风险作业报备”专题页，提供**时间筛选器**（今日/近7天/近30天/自定义）、**作业类型筛选**（动火作业/受限空间/高空作业/其他）、**状态筛选**（待审批/已审批/作业中/已完成/超时）、**企业筛选**，分为四大区块：

#### 3.1 综合统计看板（随筛选条件变化）
+ 作业报备总数
+ 作业中数 & 已完成数 & 超时未完成数
+ 平均作业时长（从计划开始到实际完成的小时数，仅统计已完成）
+ 超时作业率（超时未完成数 / 应完成作业总数 ×100%）
+ 各类作业数量占比（动火/受限空间/高空等）
+ 企业报备数量排名Top5

#### 3.2 作业报备列表（支持筛选、排序）
+ 表格字段：作业开始时间、作业类型、企业名称、作业位置、计划结束时间、实际结束时间（如有）、状态、安全负责人及电话
+ 支持按作业开始时间、状态、计划结束时间排序和筛选
+ 每条作业右侧可操作：**查看详情**（下钻到作业明细）、**提醒**（发送短信/App通知给安全负责人）

#### 3.3 单条作业明细页（点击作业详情后）
+ **作业基本信息**：
    - 报备企业、作业类型、作业位置（地图打点）
    - 计划开始/结束时间
    - 作业内容描述
    - 安全措施（如灭火器配备、通风、监护人员）
    - 作业人员名单（特种作业证号）
+ **审批记录**：
    - 审批人、审批时间、审批结果（通过/驳回）
    - 审批意见（如需增加措施）
+ **作业过程记录**：
    - 作业开始时间（实际打点或填报）
    - 作业结束时间（实际完成填报）
    - 作业现场照片/视频（可多次上传）
    - 异常情况记录（如突发状况、中断）
+ **超时提醒记录**：
    - 自动提醒时间、方式
    - 人工催办记录
+ **操作**：修改作业时间（需审批）、发送提醒、标记作业开始/结束、导出作业报告（PDF）

#### 3.4 历史趋势与效率分析（折线图/柱状图）
+ **按日/周展示**：
    - 每日风险作业报备数量趋势
    - 超时作业数量变化趋势
+ **企业分析**：
    - 各企业风险作业频次排名
    - 各企业超时作业率对比
+ **作业类型分析**：
    - 不同类型作业的超时率对比
    - 不同类型作业的平均作业时长
+ **合规性分析**：
    - 未按时报备的作业（报备时间晚于计划开始时间）的数量趋势
