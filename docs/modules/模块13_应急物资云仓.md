### 1. 模块定位
实现**应急物资台账管理**（物资名称、数量、批次、有效期、存放位置等），支持**过期预警、库存不足预警**，确保应急物资储备充足、有效可用。形成**物资入库 → 库存监控 → 领用出库 → 过期/不足预警 → 补货建议**的全生命周期管理。一级页面突出**即将过期物资、库存不足物资**，驱动值班人员及时处理（报废、采购）；二级页面支持物资台账查询、出入库记录追溯及储备分析。

+ **使用人群**：镇政府应急办/物资管理员 → 一级页面突出“即将过期物资数”和“库存不足物资数”，驱动补货或报废处理。
+ **数据显示目的**：告诉使用者“哪些物资快过期、哪些物资快用完” → 设置“紧急物资列表”，点击可查看详情并生成补货单。
+ **统计口径**：一级页面默认**所有在库物资**（不含已报废），重点监控**有效期≤30天的物资**和**库存量低于安全库存线的物资**；二级页面支持按物资类别、存放仓库、有效期筛选全部历史数据。

### 2. 一级页面核心数据（主屏概览）
| 核心指标 | 统计口径 | 展示形式 | 目的与行动含义 |
| --- | --- | --- | --- |
| 即将过期物资数 | 有效期剩余天数≤30天（可配置）的物资条目数（同一物资不同批次分别计数） | 大号橙色数字，标注“需处理” | 提醒管理员尽快安排使用或报废，避免过期浪费 |
| 库存不足物资数 | 当前库存量低于安全库存线的物资条目数 | 大号红色数字，标注“需采购” | 驱动管理员发起采购/调拨，保障应急储备 |
| 物资总数量 | 所有在库物资的总数量（按计量单位累加，如个/件/吨） | 数字（灰色背景），旁标种类数 | 了解整体储备规模 |
| 储备完好率 | 有效期内物资总价值（或数量）/ 全部物资总价值（或数量）×100%（排除过期及待报废） | 环形进度条 + 百分比（≥95%绿，85-95%橙，<85%红） | 评估物资可用性，低完好率需加强轮换 |
| 紧急物资列表 | 按**紧急程度**（已过期 > 即将过期（剩余天数升序） > 库存不足（缺口率降序））展示前5条，每条显示：物资名称、规格、存放仓库、有效期（或剩余天数）、当前库存/安全库存、状态（即将过期/库存不足） | 列表，每条右侧带“处理/采购”按钮 | 快速定位最紧急的物资，一键生成处理单或采购申请 |


> **模块标题旁**：显示“数据范围：全部在库物资”小标签，并设“查看全部台账”链接跳转二级页面。

### 3. 二级专题页面核心数据（点击下钻后）
进入“应急物资云仓”专题页，提供**物资类别筛选**（防汛物资/消防物资/医疗物资/生活物资/其他）、**仓库筛选**（中心库/村级库/企业代储点等）、**状态筛选**（正常/已过期/库存不足/即将过期），分为四大区块：

#### 3.1 综合统计看板（随筛选条件变化）
+ 物资条目总数（按批次数）
+ 物资种类数（按物资名称去重）
+ 物资总数量（按计量单位）
+ 即将过期物资数 & 已过期物资数 & 库存不足物资数
+ 储备完好率（有效期内数量占比）
+ 各仓库物资价值/数量分布
+ 各物资类别数量占比

#### 3.2 物资台账列表（支持筛选、排序）
+ 表格字段：物资名称、规格型号、类别、存放仓库、批次号、有效期、当前库存、安全库存、单位、状态（正常/即将过期/已过期/库存不足）、入库时间
+ 支持按有效期、库存量、类别、仓库排序和筛选
+ 每条物资右侧可操作：**查看详情**（下钻到物资明细及出入库记录）、**处理**（过期物资报废/退换）、**采购申请**（库存不足）

#### 3.3 物资明细页（点击物资详情后）
+ **物资基本信息**：
    - 物资名称、规格型号、类别、单位
    - 存放仓库及具体货位
    - 批次号、生产日期、有效期
    - 当前库存、安全库存、最大库存（可选）
    - 供应商/采购来源
+ **库存变动记录**（按时间倒序列表）：
    - 操作时间、操作类型（入库/出库/调拨/报废）、数量、经办人、备注
+ **预警记录**：
    - 过期预警记录（预警时间、处理状态）
    - 库存不足预警记录（预警时间、采购申请是否发起）
+ **操作**：入库登记、出库登记、调拨、报废、修改安全库存、生成采购申请、打印物资标签

#### 3.4 历史趋势与效率分析（折线图/柱状图）
+ **按月展示**：
    - 库存总量变化趋势
    - 即将过期物资数量变化（建议关注季节性或突发需求）
+ **物资类别分析**：
    - 各类物资的库存占比及完好率
    - 各类物资的周转率/使用频率
+ **仓库分析**：
    - 各仓库的物资种类数、总量、完好率
    - 各仓库的过期/不足问题统计
