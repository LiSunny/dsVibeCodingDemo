# 模块4：重点人员联勤

### 1. 模块定位
建立**独居老人、残障人士、涉稳人员、刑满释放人员**等重点人员档案，支持**走访任务制定 → 打卡执行 → 漏访统计预警**的闭环管理，确保重点人员不脱管、不漏访。一级页面突出**今日应访人数、当前漏访人数**以及**漏访预警**，驱动值班人员或网格员及时完成走访任务；二级页面支持人员档案管理、走访历史追溯、漏访分析及走访效率评估。

+ **使用人群**：镇政府综治办/网格管理中心人员、网格员 → 一级页面突出"漏访人数"和"今日未访人数"，驱动立即安排走访或催办。
+ **数据显示目的**：告诉使用者"哪些人员今天需要走访、哪些已经漏访" → 设置"最新漏访/未访人员列表"按漏访天数排序，支持一键派发走访任务。
+ **统计口径**：一级页面默认**今日及历史未完成的走访任务**（今日应访未访 + 历史应访未访即漏访），以及未来7天应访任务预览；二级页面支持按人员类型、区域、走访状态筛选全部历史数据。

---

### 2. 一级页面核心数据（主屏概览）
| 核心指标 | 统计口径 | 展示形式 | 目的与行动含义 |
| --- | --- | --- | --- |
| 今日未访人数 | 今日应完成走访（按走访计划）但尚未打卡走访的人数 | 大号橙色数字，标注"需今日完成" | 告知值班人员/网格员今日还有多少重点人员未走访，需立即执行 |
| 累计漏访人数 | 应访日期已过（超过计划日期）但至今未完成走访的人数（不限时间） | 大号红色数字，标注"严重漏访" | 最高优先级，需追访或重新安排走访，防止脱管 |
| 今日应访总数 | 今日计划应完成走访的人数（按走访任务配置，如独居老人每周一次） | 数字（灰色背景），旁标环比昨日 | 了解当日走访工作量 |
| 走访覆盖率（近7天） | 近7天内实际完成走访人数 / 近7天内应访总人数 ×100% | 环形进度条 + 百分比（≥95%绿，80-95%橙，<80%红） | 评估近期走访工作的整体覆盖情况 |
| 漏访/未访人员列表 | 按**优先级**（累计漏访 > 今日未访）展示前5条，每条显示：人员姓名、类型（独居老人/残障/涉稳/刑释）、所属村/社区、应访日期、漏访/未访天数（今日未访显示"今日需访"） | 列表，每条右侧带"派发任务/催办"按钮 | 快速定位漏访或今日未访人员，一键派发给网格员或催办 |


> **模块标题旁**：显示"数据范围：今日应访+累计漏访"小标签，并设"查看全部历史"链接跳转二级页面。

### 3. 二级专题页面核心数据（点击下钻后）
进入"重点人员联勤"专题页，提供**时间筛选器**（今日/近7天/近30天/自定义）、**人员类型筛选**（独居老人/残障/涉稳/刑满释放）、**走访状态筛选**（已走访/未走访/漏访）、**区域筛选**（村/社区），分为四大区块：

#### 3.1 综合统计看板（随筛选条件变化）
+ 重点人员总数（分类型统计）
+ 应走访总人次（时间范围内计划走访次数）
+ 实际走访人次 & 漏访人次
+ 走访完成率 = 实际走访人次 / 应走访总人次 ×100%
+ 平均走访响应时长（从应访日期到实际走访的天数，仅统计已走访）
+ 漏访人员数 & 漏访率（漏访人数 / 应访总人数）
+ 各类人员走访完成率对比

#### 3.2 人员档案列表（支持筛选、排序）
+ 表格字段：姓名、性别、年龄、类型（独居老人/残障/涉稳/刑释）、所属村/社区、应急联系人及电话、走访计划（周期）、最近走访时间、当前状态（已访/未访/漏访）、责任人（网格员）
+ 支持按类型、状态、最近走访时间排序和筛选
+ 每条人员右侧可操作：**查看详情**（下钻到人员档案及走访记录）、**派发任务**（制定走访任务）、**催办**（发提醒给责任人）

#### 3.3 人员档案及走访明细页（点击人员详情后）
+ **人员基本信息**：
    - 姓名、性别、身份证号、出生日期、年龄
    - 人员类型（独居老人/残障/涉稳/刑满释放）
    - 居住地址（地图打点）、所属村/社区
    - 应急联系人姓名、关系、联系电话
    - 备注（健康状况、特殊需求等）
+ **走访计划配置**：
    - 走访周期（每周/每半月/每月）
    - 最近一次计划日期
    - 下次应访日期
+ **走访记录**（按时间倒序列表）：
    - 计划走访日期、实际走访日期、走访人员（网格员）、打卡时间、走访内容/备注、走访照片（可选）
    - 漏访标记（如计划日期已过但无实际走访）
+ **漏访预警记录**：
    - 漏访日期、预警时间、提醒方式、处理结果
+ **操作**：编辑人员档案、制定走访任务（单次/周期）、手动补录走访、导出人员档案（PDF/Excel）

#### 3.4 历史趋势与效率分析（折线图/柱状图）
+ **按日/周展示**：
    - 每日应访人数 vs 实际走访人数 vs 漏访人数
    - 走访完成率趋势
+ **类型分析**：
    - 各类人员的走访完成率对比
    - 各类人员的漏访率趋势
+ **区域分析**：
    - 各村/社区走访完成率排名
    - 各网格员走访任务完成率排名
+ **漏访原因分析**（需标签统计）：任务过多/人员外出/网格员忘记等
