# 模块5：主体安全履责

### 1. 模块定位
以**企业为基本单元**，通过**履职卡（隐患排查）打卡**机制，考核企业安全主体责任落实情况。重点监控**履职完成率（隐患排查完成率）**，对**非100%完成的企业**进行异常预警，形成**打卡任务发布 → 企业履职填报 → 完成率统计 → 异常督办**的管理闭环。一级页面突出**履职未达标企业数、整体完成率分布**及**异常企业列表**，驱动值班人员督促企业按时完成隐患排查；二级页面支持企业履责趋势分析、履职明细追溯及排名考核。

+ **使用人群**：镇政府安监站/值班人员 → 一级页面突出“履职未达标企业数”，驱动督促企业完成隐患排查打卡。
+ **数据显示目的**：告诉使用者“哪些企业未完成履职打卡，需督办” → 设置“履职异常企业列表”按完成率升序排列，支持一键发送提醒。
+ **统计口径**：一级页面默认**近7天（或本月）应完成的履职任务**，以企业为单位统计**履职完成率**：完成率 = 该企业已完成打卡次数 / 应完成打卡次数 ×100%。（应完成次数根据企业风险等级、行业要求等配置）。**仅当完成率≠100%时视为异常**；二级页面支持按时间周期（周/月/季度）、企业行业、风险等级筛选全部历史数据。

### 2. 一级页面核心数据（主屏概览）
| 核心指标 | 统计口径 | 展示形式 | 目的与行动含义 |
| --- | --- | --- | --- |
| 履职未达标企业数 | 统计周期（默认近7天/本月）内，履职完成率 ≠100%的企业总数（即至少漏打卡1次） | 大号红色数字，标注“需督办” | 告知值班人员当前有多少企业未完成全部履职任务，需立即沟通或督促 |
| 整体履职完成率 | 统计周期内，所有企业已打卡总次数 / 应打卡总次数 ×100% | 环形进度条 + 百分比（≥95%绿，80-95%橙，<80%红） | 宏观评估辖区企业安全履责整体水平，判断是否需要集中整治 |
| 完成率分布 | 按企业完成率区间统计：100%完成企业数、80%-99%企业数、<80%企业数 | 柱状图或饼图 | 快速识别完成率较低的企业群体，便于分类管理 |
| 高风险企业完成率 | 风险等级为“高”或“重大”的企业，其平均履职完成率 | 百分比 + 趋势箭头 | 重点监管高风险企业，确保其隐患排查无遗漏 |
| 履职异常企业列表 | 按**完成率升序**（完成率低的在前）展示前5条异常企业（完成率<100%），每条显示：企业名称、行业、风险等级、应打卡次数、实际打卡次数、完成率 | 列表，每条右侧带“提醒”按钮 | 快速定位履职最差的企业，一键发送提醒或约谈 |


> **模块标题旁**：显示“统计周期：近7天（可配置）”小标签，并设“查看全部历史”链接跳转二级页面。

### 3. 二级专题页面核心数据（点击下钻后）
进入“主体安全履责”专题页，提供**时间周期选择器**（本周/本月/本季度/自定义）、**企业筛选**（按行业、风险等级、区域）、**完成率区间筛选**，分为四大区块：

#### 3.1 综合统计看板（随筛选条件变化）
+ 企业总数（纳入考核的企业数量）
+ 应打卡总次数
+ 实际打卡总次数
+ 整体履职完成率
+ 未达标企业数（完成率<100%的企业数）及占比
+ 平均完成率（所有企业的完成率平均值）
+ 高风险企业数量 & 其平均完成率
+ 完全达标企业数（完成率=100%） & 零打卡企业数（完成率=0%）

#### 3.2 企业履职列表（支持筛选、排序）
+ 表格字段：企业名称、行业、风险等级、应打卡次数、实际打卡次数、完成率、最近打卡时间、是否达标（是/否）、企业负责人及联系方式
+ 支持按完成率、应打卡次数、风险等级排序和筛选
+ 每条企业右侧可操作：**查看详情**（下钻到企业履职明细）、**发送提醒**（督促企业完成打卡）

#### 3.3 企业履职明细页（点击企业详情后）
+ **企业基本信息**：
    - 企业名称、统一社会信用代码、行业、风险等级、地址
    - 安全负责人姓名、联系电话
+ **履职任务配置**：
    - 考核周期（如每周/每月）
    - 应打卡任务清单（如“每周一次消防自查”“每月一次设备巡检”等，可配置）
+ **打卡记录**（按时间倒序列表）：
    - 打卡时间、打卡任务名称、打卡人、打卡凭证（照片/签名/GPS位置）
    - 异常标记（如超时打卡、漏打卡）
+ **履职完成率趋势**（折线图）：
    - 按月/周展示该企业历史完成率变化
+ **操作**：配置考核周期、手动补录打卡、发送提醒、导出企业履职报告（PDF/Excel）

#### 3.4 历史趋势与效率分析（折线图/柱状图）
+ **按周/月展示**：
    - 整体履职完成率变化趋势
    - 未达标企业数量变化
    - 高风险企业完成率 vs 低风险企业完成率对比
+ **行业/区域分析**：
    - 各行业企业平均完成率排名
    - 各村/社区内企业完成率分布
+ **企业排名**：
    - 履职完成率最高/最低的企业Top10
    - 连续多周期未达标企业名单（需重点关注）
